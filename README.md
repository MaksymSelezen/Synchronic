# Synchronic

## Інструкція запуску

1. Клонувати репозиторій:

```
git clone https://github.com/MaksymSelezen/Synchronic.git
```

2. Перейти в папку проєкту:

```
cd Synchronic
```

3. Встановити залежності:

```
npm install
```

4. Запустити проєкт локально:

```
npm run dev
```

5. Відкрити в браузері посилання, яке зʼявиться в терміналі.

## Збірка проєкту

```
npm run build
```

## Перегляд зібраного проєкту

```
npm run preview
```
